# English AI Teacher

ブラウザだけで動く英会話練習アプリです。GitHub Pages で配信しています。

## 公開URL

設定後、Settings → Pages に表示されるURLに置き換えてください。
（Organizationリポジトリの場合: `https://<org名>.github.io/<repo名>/`）

## 初回利用時の設定

1. アプリ右上の「設定」タブを開く
2. Gemini APIキーを入力（[Google AI Studio](https://aistudio.google.com/apikey) で取得）
3. APIキーはこの端末の `localStorage` にのみ保存され、サーバーには送信されません

## 注意事項

- このリポジトリはPublicです。コードは誰でも閲覧できますが、APIキーはコードに含まれていません。
- Google AI Studio側でAPIキーに「HTTPリファラー制限」と「有効期限」を設定することを推奨します。
